&par Example Description

This example shows how to configure the DAC peripheral to generate NoiseWave
The waveform can be displayed using an oscilloscope. using DAC_CHANNEL_1(PA4)
to output NoiseWave.

&par Directory contents

  - DAC/NoiseWave/src/apm32f10x_int.c     Interrupt handlers
  - DAC/NoiseWave/src/main.c                     Main program


&par Hardware and Software environment

  - This example runs on APM32F103_MINI Devices.
